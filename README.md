Задача: Генерация рекомендаций по удобрениям на основе состояния поля
Описание: В данной работе проведено исследование по генерации рекомендуемых доз N, P, K с условиями состояния почвы, погоды и прошлого урожая.

Целью данной работы является Создать доступный инструмент для агрономов и фермеров, позволяющий рассчитывать научно обоснованные дозы удобрений,
учитывать множество факторов (почва, погода, предшественник и визуализировать ожидаемый экономический эффект

Мотивация данной работы является экономическая и экологическая состовляющие. Экономическая: Перерасход удобрений ведет к лишним затратам, экологическая: недостаток — к недобору урожая

Данные: Agricultural Yield and Fertilizer Use (FAO) (https://www.fao.org/faostat/en/#data/RF)

Архитектура модели и обоснование выбора:
В проекте не используется машинное обучение, а применяется классический балансовый метод агрохимии, так как:

1) Существуют устоявшиеся научно обоснованные нормативы

2) Данные для ML (тысячи полевых опытов) в открытом доступе отсутствуют

3) Детерминированная модель прозрачна и интерпретируема

Формула расчета дозы удобрений:
Доза = (Вынос - Доступно_из_почвы - Кредит_предшественника) × Поправки / КПД_удобрений
